## Install

```shell-session
$ go get -u github.com/ntrrg/sport-calc/...
```

## Usage

```shell-session
$ sport-calc [-p :PORT]
```

## Uninstall

```shell-session
$ go clean -i github.com/ntrrg/sport-calc/...
```

```shell-session
$ rm -rf $GOPATH/github.com/ntrrg/sport-calc
```

```shell-session
$ rm -rf $GOPATH/pkg/$(go env GOHOSTOS)_$(go env GOHOSTARCH)/github.com/ntrrg/sport-calc
```
